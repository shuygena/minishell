# minishell

Lost in The Shell.

## :toolbox: Requirements:  
GCC compiler  
GNU Make

## :link: Download & Compile
Clone repository:
```
git clone https://github.com/shuygena/minishell minishell
```
Go to directory:
```
cd minishell/minishel\ v01/
```
For compile program run:   
```
make
```  

## :pager: How to use
Launch it with:
```
  ./minishell
```
Use it like bash)
